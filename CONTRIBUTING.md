# Contributing to Awesome MBTI

Thanks for considering a contribution. This list aims to be the most useful single starting point for serious MBTI exploration. Quality matters more than volume.

## What we accept

- **Resources with clear standalone value**: tests, profile sites, books, academic papers, communities, tools, datasets.
- **Critical perspectives**: academic critiques and journalism that helps readers evaluate MBTI honestly. Neutrality is a feature.
- **Resources accessible to readers**: free public content, books available for purchase, clearly-priced paid tools.

## What we reject

- **Pure marketing or affiliate links** with no underlying substance.
- **Single blog posts** unless they're widely-cited foundational pieces (e.g., Adam Grant's MBTI critique).
- **Resources without clear authorship or attribution** (anonymous personality calculators, content-farm sites).
- **Resources duplicating something already listed** (we list one canonical version per resource).
- **Self-promotion-only accounts** (PR purely to add an unrelated personal site).
- **Resources behind paywalls without clear value indication** (acceptable for established academic books; not for new subscription products with no preview).

## Entry format

Each line in the README must follow this format exactly:

```
- [Resource Name](https://url-here) — One-line description in plain language, no marketing voice.
```

Specific rules:
- Use an em dash (`—`) not a hyphen between name and description.
- Description: factual, neutral, ≤ 25 words. Describe what the resource IS, not why it's "best."
- No "the most popular" / "the leading" — let readers judge.
- No emojis in descriptions.
- Alphabetical order within each section.

## How to submit

1. Fork this repository.
2. Add your entry in the appropriate section, preserving alphabetical order.
3. Open a pull request with title format: `Add: <Resource Name> to <Section>`.
4. PR description should include: (a) one paragraph on why this resource fits the section, (b) any conflicts of interest (e.g., "I'm the author of this resource").

## What happens after submission

- Maintainers review within 7 days.
- Resources fitting clear scope are merged quickly.
- Borderline resources may be discussed in the PR thread before merging.
- Rejections come with a one-sentence reason; you're welcome to revise and resubmit.

## Reporting issues

If you find a broken link, outdated description, or a resource that no longer meets the criteria above, please open an issue. We treat list maintenance as a continuous responsibility, not a one-time launch.

## Code of conduct

Be respectful in PR discussions. Critical perspectives on listed resources are welcome; personal attacks on contributors or maintainers are not. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).
